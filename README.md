# 行知
   工程名（flutter_todo_list）待办事项app。
   版本0.1.3
    
    该项目是一个多级分类归档的备忘录项目。可以通过对待办事项添加标签和项目分属进行整理，并设置事件进 行提醒，有云同步功能。项目客户端使用Flutter开发，以此提供跨平台能力。服务器使用Django开发，作为应 用后端支持，使用Nginx+uWSGI+MySQL部署在云服务器提供全天候响应工作。
